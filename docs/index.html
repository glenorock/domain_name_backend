Testing synchronous functions

Testing asynchronous functions

During the test of asynchronous functions with jasmine, certain problems occured as the later seemed to ignore the async and await tags there by ending the testing before time. The test files where hence made using tests directory, and the launching of the later shall be done with the command node tests/name_of_the_file.js
